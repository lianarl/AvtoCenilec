# AvtoCenilec

Aplikacija za ocenjevanje tržne vrednosti vozil.

## Zahteve

- Node.js 18+ (priporočeno Node.js 20)
- npm

## Lokalni zagon

```bash
npm install
npm run dev
```

Teče na `http://localhost:8080`.

## Produkcijska gradnja

```bash
npm run build
```

## Testiranje

```bash
npm run test
```
